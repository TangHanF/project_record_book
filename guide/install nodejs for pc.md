# Windows系统安装NodeJS

## 下载NodeJS

[点击此处下载](https://nodejs.org/en/download/)

![](https://gitee.com/TangHanF/pic/raw/master/43dlsG.png)

> 如果自己系统是64位的请下载64位版本NodeJS，如果是32位请下载32位NodeJS

## 验证安装是否成功

> 打开cmd窗口，输入：

```bash
node -v
```

安装成功则输出以下内容：

顺便验证以下npm版本：

